# Admin — Desktop Only

O painel administrativo (`/admin/*`) é otimizado para uso em **desktop com mouse**.

- **TV Box / Controle remoto:** Não há suporte a navegação D-pad (`data-nav-*`). O admin não foi adaptado para uso em Android TV ou controle remoto.
- **Uso recomendado:** Acesse o admin em um navegador desktop ou tablet com teclado/mouse.

Para adaptar o admin para TV Box no futuro, seria necessário adicionar `data-nav-row` e `data-nav-item` nos botões e tabelas principais, além de handlers de teclado para setas e Enter.
